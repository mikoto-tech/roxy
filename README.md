# Roxy - 洛琪希

一套强大的Java爬虫框架，其主要框架设计来自于`Mikoto-Pixiv`。

## 项目构思
Roxy提供是简易，便于管理的现代化数据爬虫

#### 为了使用Roxy您需要做什么？
- 规范数据模型

是的，仅此而已！当然，不可避免地，当此工作所需超出Roxy的数据解析工具集之时，您仍然需要自行编写部分类。

## 技术选型
- 数据库连接池：Druid
- 线程池：hippo4j
